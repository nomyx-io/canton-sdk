## [0.1.0] - 2024-12-01

### Added
- Initial release
- CantonClient with create, exercise, and query commands
- Auth providers: Sandbox, OAuth2, and Static token
- Retry logic with configurable exponential backoff
- Health check endpoint support for participant and ledger API connectivity
